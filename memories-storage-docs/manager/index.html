    convert_image

    def convert_image(src_path: Union[str, Path, FileStorage, BytesIO],
                  dest_path: str,
                  format: Optional[str] = None,
                  width: Optional[int] = None,
                  height: Optional[int] = None,
                  quality: int = 85,
                  fit_mode: ImageFitMode = ImageFitMode.CONTAIN,
                  keep_exif: bool = False) -> str

    Convert an image to the specified format and resize it based on fit_mode.

    Optionally, preserve EXIF metadata if requested.

    Arguments:

    • src_path: Source image file path
    • dest_path: Destination file path where converted image will be saved
    • format: Target format (e.g., 'JPEG', 'PNG', 'WEBP', 'BMP', etc.)
    • width: Desired width of the bounding box (optional)
    • height: Desired height of the bounding box (optional)
    • quality: JPEG compression quality (default: 85)
    • fit_mode: How the image should fit within the bounding box (ImageFitMode.CONTAIN or ImageFitMode.COVER)
    • keep_exif: Whether to retain EXIF metadata (only for supported formats)

    Returns:

    Path to the converted image

    Manages resource_id assignments for a given resource type. - Stores resource_id as a set for optimal lookups and modifications. - Provides a thread-safe structure using threading.Lock.

    Arguments:

    • resource_name str - The type of the resource (e.g., 'books', 'documents').

    generate_resource_id

    def generate_resource_id(user_id: str) -> str

    Generates a unique resource ID using UUID.

    Arguments:

    • user_id str - The ID of the user who owns the resource.

    Returns:

    • str - A newly generated unique resource ID.

    Process: 1. Acquire thread lock for safe concurrent modifications. 2. Generate a new UUID-based resource ID. 3. Store the new resource ID in the set. 4. Return the newly created resource ID.

    release_resource_id

    def release_resource_id(user_id: str, resource_id: str) -> None

    Deletes a resource ID for a user.

    Arguments:

    • user_id str - The ID of the user who owns the resource.
    • resource_id str - The unique identifier for the resource.

    Process: 1. Acquire thread lock for safe concurrent access. 2. Remove the specified resource_id from the set.

    class ContentIdManager()

    Manages content_id assignments for a given resource type. - Maintains a persistent set of content IDs per user and resource. - Ensures thread-safe operations using threading.Lock.

    __init__

    def __init__(resource_name: str, storage_backend: AbstractStorageBackend)

    Initializes the content ID manager.

    Arguments:

    • resource_name str - The type of the resource (e.g., 'books', 'documents').
    • storage_backend AbstractStorageBackend - The backend used for storage operations.

    generate_content_id

    def generate_content_id(user_id: str, resource_id: str) -> int

    Generates the next available content_id within a resource.

    Arguments:

    • user_id str - The ID of the user who owns the resource.
    • resource_id str - The unique identifier for the resource.

    Returns:

    • int - The next available content_id.

    Process: 1. Acquire thread lock for safe concurrent access. 2. Load existing IDs and determine the next available ID. 3. Prioritize IDs from 1 to 9, then find the lowest unused integer. 4. Add the assigned content_id to the set and return it.

    release_content_id

    def release_content_id(user_id: str, resource_id: str, content_id: int)

    Releases a content_id when content is deleted.

    Arguments:

    • user_id str - The ID of the user who owns the resource.
    • resource_id str - The unique identifier for the resource.
    • content_id int - The content ID to be released.

    Process: 1. Acquire thread lock for safe concurrent modifications. 2. Remove the specified content_id from the active set.
